\doxysubsection{Detailed Description}
Represent game state. 

This class is used to represent the state of a game being implemented. It is a base class with virtual functions, and all game implementations should inherit from it.

It holds information about\+: The current board state Whose turn it is Any additional data required to determine who is winning/what point the game is at
