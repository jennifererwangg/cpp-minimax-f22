\doxysubsection{Detailed Description}
The implementation of the minimax algorithm. 

This class contains the acutal implementation of the minimax algorithm. It is game-\/agnostic, as it only calls functions on the \mbox{\hyperlink{classminimax_1_1core_1_1_game_state}{Game\+State}} base class, which all of the implemented games rely on.
